What we've learned so far:

  1. Nested Lists
    1. This page consists of multiple nested lists.
    2. Nested lists are lists that exist within other lists.
    3. Nested lists can make it easier to organize content, and they aren't required to share the same style type as their parent list.
  2. Semantic Tags
    1. Semantic tags are used to group data together in order to improve its logical structure.
    2. Semantic tags can consist of tags, such as header, navigation, main, and footer.
    3. These tags are block level tags that work with CSS to improve a page's visual format.
  3. CSS Syntax
    1. The syntax for CSS uses if/then rules and consists of two parts, selectors and declarations.
    2. If the computer finds the selector to be a match for the data, it'll execute the following declarations.
    3. For example: h1 { color: blue; font-size: 28 pt; }. In this case, the selector is "h1" and the declarations are "color: blue;" and "font-size: 28 pt;". If the data on the page is an h1, the computer will change the data's color and font size accordingly.
